# NovaVault.io

**Ledger-First Loyalty OS — Web2 now, Web3-ready later.**

NovaVault is a loyalty operating system that replaces opaque "points math" with a transparent, auditable ledger. It layers commerce, campaigns, analytics, and fraud controls on top — starting as a Web2 SaaS platform with an optional Cardano token upgrade path.

## This Repository

This repo contains the **marketing and pitch site** for NovaVault.io — not the application itself. It includes:

- Landing page with features, pricing, roadmap, and contact form
- Founder's Statement of Purpose
- Complete investor-ready business plan (HTML + PDF)
- Team CVs

## Tech Stack

- Static HTML + vanilla CSS + vanilla JS (no framework, no build step)
- Hosted on Apache shared hosting (BlueHost/cPanel)

## Deployment

Upload files directly to the server. No build step required.
